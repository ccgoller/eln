# ⚙️ Instructor Setup Guide — Electronic Lab Notebook on GitHub

This guide walks you through setting up the GitHub-based ELN system for your lab or course, from creating the organization to reviewing student notebooks at the end of the term.

## 🔧 Prerequisites

Before you begin, you will need:

- A GitHub account. If you do not have one, create one at [github.com](https://github.com).
- A list of student GitHub usernames or email addresses (collect these in advance, e.g., via a course survey).
- Approximately **30–60 minutes** to complete the full setup.
