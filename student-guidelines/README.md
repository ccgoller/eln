## Quick Reference

| Task | How |
|------|-----|
| Write a new entry | Copy `templates/experiment-entry.md`, fill it in, commit |
| Save your work | `git add`, `git commit -m "message"`, `git push` |
| Add a data file | Place file in `data/`, commit with a descriptive message |
| Write a weekly summary | Copy `templates/weekly-summary.md`, fill it in, commit |
| Ask for help | Open a GitHub Issue in your notebook repository |
